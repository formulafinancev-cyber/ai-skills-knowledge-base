## Repository Evaluator
This repo includes `/evaluate-repository.md` — a rigorous security evaluation prompt for reviewing any Claude Code skill/plugin before installing. Checks: implicit execution surfaces, trust boundaries, hooks that auto-execute, network access patterns, persistent state files. Use before installing unknown skills.
