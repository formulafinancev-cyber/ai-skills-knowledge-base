# BEGINNER DEPLOYMENT REQUIREMENTS

## Purpose
This file defines the beginner-friendly deployment requirement for the Formula Finance AI Office project.

Its purpose is to ensure that all future implementation and deployment instructions are written for a user with minimal server and infrastructure knowledge.

## Main Principle
Implementation guidance must be understandable for a beginner.

The project should assume that the user may not yet know:

- how to deploy to a server;
- how to configure environment variables;
- how to connect APIs;
- how to install dependencies;
- how to run services;
- how to debug infrastructure issues.

## Instruction Style Rule
All future build and deployment instructions should be written in a way that explains:

- what is being done;
- why it is being done;
- where the user gets the required value or credential;
- where exactly it should be placed;
- what result should appear if the step worked.

## Required Beginner Detail
Instructions should not stop at high-level phrases like:
- connect Telegram API;
- configure MCP;
- deploy to server;
- set env variables.

Instead, they should explain:
- where to create the token or credential;
- what menu/button to click;
- what file to open;
- what line to edit;
- what command to run;
- what output or screen to expect.

## Deployment Explanation Requirement
When deployment is described, it should include:

- server prerequisites;
- recommended server type;
- required software to install;
- exact sequence of commands;
- how to start the service;
- how to restart the service;
- how to check logs;
- how to know the service is running.

## Integration Explanation Requirement
When an API, MCP, bot, or external service is introduced, instructions should include:

- what the service is;
- why the project needs it;
- what account or access is required;
- how to obtain credentials;
- where to put those credentials;
- how to test the connection.

## Beginner Safety Rule
The project should avoid assuming hidden knowledge.

If a step could confuse a beginner, it should be explained explicitly.

## Final Principle
The Formula Finance AI Office should be implementable and deployable by a beginner through detailed, step-by-step instructions written in plain language.
